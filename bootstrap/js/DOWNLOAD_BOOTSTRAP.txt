DOWNLOAD BOOTSTRAP JAVASCRIPT HERE
====================================

You need to download Bootstrap 5.3.0 JavaScript file and place it in this folder.

Download Link:
https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js

Steps:
1. Go to the link above
2. Right-click and "Save As" OR just copy the entire content
3. Create a file named "bootstrap.bundle.min.js" in this folder
4. Paste the content

OR use this command in PowerShell (if you have internet):
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js" -OutFile "bootstrap.bundle.min.js"

Once downloaded, the website will work fully offline!
