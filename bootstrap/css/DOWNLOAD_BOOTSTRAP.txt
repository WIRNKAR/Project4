DOWNLOAD BOOTSTRAP CSS HERE
=============================

You need to download Bootstrap 5.3.0 CSS file and place it in this folder.

Download Link:
https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css

Steps:
1. Go to the link above
2. Right-click and "Save As" OR just copy the entire content
3. Create a file named "bootstrap.min.css" in this folder
4. Paste the content

OR use this command in PowerShell (if you have internet):
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" -OutFile "bootstrap.min.css"

Once downloaded, the website will display properly!
