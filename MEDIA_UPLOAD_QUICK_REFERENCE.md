# Quick Reference: Media Upload System

## 🚀 Quick Start

### Add Image (30 seconds)
1. Admin Dashboard → "Upload New Image" section
2. Enter: Title + Price + Select Image
3. Click "Add Item to Gallery"
✅ Done! Image appears on website instantly

### Add Video from File (1-5 minutes)
1. Admin Dashboard → "Upload New Video" section
2. Enter: Title + Description + Select Video File
3. Click "Add Video Now"
✅ Done! Video streams from your website

### Add Video from YouTube (30 seconds)
1. Admin Dashboard → "Upload New Video" section
2. Enter: Title + Description + YouTube URL
3. Click "Add Video Now"
✅ Done! Video embedded on your website

## 🔔 Status Indicators

**Gallery Status (Card Header):**
- 🟢 **Green dot** = 0-3 items (plenty space)
- 🟡 **Yellow dot** = 4-5 items (getting full)  
- 🔴 **Red dot** = 6 items (full, need to replace)

**Upload Status:**
- ⏳ "Uploading X%" = In progress
- ✅ "Success" = Complete, visible now
- ❌ "Error" = Failed, try again
- ℹ️ "Info" = Important note

## ⚡ Performance Tips

### Optimize Before Upload

**For Images:**
```
Recommended: 1200 x 900 pixels
Quality: 85% (good balance)
Format: JPG for photos, PNG for graphics
```

**For Videos:**
```
Recommended: 1080p (1920×1080)
Bitrate: 5-8 Mbps
Frame rate: 24-30 FPS
Duration: 30 sec - 3 min
```

### Tools to Compress Files

**Images:**
- TinyPNG.com (free)
- ImageOptimizer (online)
- Photoshop Export

**Videos:**
- HandBrake (free, desktop)
- CloudConvert (free, online)
- VidCompact (mobile)
- YouTube's own compression

## 🎬 Video Format Guide

### Local Video Upload
**Best for:** Exclusive content, private videos
**Requirements:** MP4, WebM, OGG format
**Process:** File uploaded to local storage
**Playback:** Streams from your server

### YouTube Embed
**Best for:** Public YouTube videos, playlists
**Requirements:** Valid YouTube URL
**Process:** Video linked to public YouTube
**Playback:** Streams from YouTube servers

### Choose Based On:
| Need | Use | Reason |
|------|-----|--------|
| Large audience | YouTube | Better streaming |
| Exclusive content | Local | Only on your site |
| Already on YouTube | YouTube | No re-upload |
| Privacy control | Local | Full control |
| Save bandwidth | YouTube | YouTube handles it |

## 🛡️ Safety & Security

✅ **Data Safety:**
- All uploads stored locally in your browser/device
- Not sent to external servers (except YouTube)
- Password-protected files recommended
- Regular backups suggested

✅ **File Safety:**
- Only image/video files accepted
- Malicious files rejected
- Size limits prevent abuse
- Format validation required

## 📞 Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| "Gallery Full" | Delete old item or accept auto-replace |
| "File Too Large" | Compress file using tools above |
| "Upload Stuck" | Refresh page, check connection |
| "Can't Save Price" | Only use numbers, no symbols |
| "Video Won't Play" | Check format, re-upload |
| "Changes Not Showing" | Hard refresh (Ctrl+Shift+R) |
